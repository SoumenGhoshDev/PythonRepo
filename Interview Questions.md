# Interview Questions

### Is python a Dynamically Typed Language ? Why it is called so?
Yes, We don't have to explicitly mention the variable Data type. It will directly assume the data type from the syntax in the time of assigning a value into it.
  
### How to put multiline comment in Python?

Use `"""` to put Multiline Comment.
``` Python
"""
This is
a
Multi-Line
Comment
"""
```
